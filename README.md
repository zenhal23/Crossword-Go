# Crossword GO

A turn-based crossword puzzle game for Android, built in Unity 6.

## What to expect

- A crossword grid where two players (you vs a bot) take turns placing letters
- Drag letters from your hand onto the grid, submit your turn, and score points for correct placements
- Three bot difficulty levels — Easy, Medium, and Hard
- Levels built with a custom Unity Editor level editor tool, supporting text and image clues
- Levels cycle from a shared library with progress saved across sessions

## Tech

- Unity 6000.3.8f1 · URP 2D · Android
- IL2CPP · ScriptableObject-based data layer
- NUnit edit-mode tests for core systems

## Status

Prototype in active development.
